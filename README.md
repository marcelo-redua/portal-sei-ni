# portal-sei-ni

## Descrição

Este é o repositório do código-fonte para o Portal do Sistema Eletrônico de Informações (SEI) da Prefeitura de Nova Iguaçu. O portal foi criado para facilitar o acesso dos cidadãos às informações e serviços municipais.

## Recursos

- Acesso rápido a informações e serviços
- Interface amigável ao usuário
- Compatível com todos os dispositivos modernos
